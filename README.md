# gomap-concurrent

Concurrent general-purpose shared map for using with flexible data.

## Import

Import the package:

```go
import (
	gmap "github.com/kolobovdg/gomap-concurrent"
)
```
```bash
go get "github.com/kolobovdg/gomap-concurrent"
```

Running tests:
```bash
go test "github.com/kolobovdg/gomap-concurrent"
```
